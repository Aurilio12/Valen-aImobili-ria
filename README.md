# Valença Imobiliária - CDE

Site estático premium e responsivo para catálogo de imóveis em Ciudad del Este.

## Páginas
- `index.html` (home)
- `catalogo.html` (catálogo com filtros)
- `imovel.html` (detalhes do imóvel)
- `sobre.html`
- `contato.html`

## Publicação
Você pode publicar em qualquer hospedagem estática (Netlify, Vercel, GitHub Pages, Cloudflare Pages ou cPanel).

Basta enviar todos os arquivos deste diretório mantendo a estrutura de pastas.
